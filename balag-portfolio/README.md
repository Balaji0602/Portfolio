# BalaG Portfolio

A modern, responsive, and animated personal portfolio website built with React and Vite.

## Features

- **Dynamic Theming**: Light and Dark mode support with customized gradient aesthetics and background effects (Floating Orbs).
- **Responsive Navigation**: A sticky, glassmorphism-inspired navigation bar that adjusts seamlessly between desktop and mobile views.
- **Smooth Animations**: Powered by `framer-motion` for scroll-reveal animations, animated skill bars, staggered grid entries, and floating UI elements.
- **Modern UI/UX**: Designed using TailwindCSS for utilities, alongside custom CSS variables. Clean typography and carefully selected color palettes.

## Sections Included

1. **Hero**: High-impact introduction highlighting the "Full-Stack Developer" role, experience metrics, available opportunities badge, and highly interactive floating avatar element.
2. **About**: Details professional background (Technical Lead at Zeb, formerly Software Engineer at Avasoft), educational qualifications, and key accomplishments.
3. **Skills**: Categorized technology summary (Frontend, Backend, Database, Cloud, Programming, Core CS, AI) visualized with animated progress bars.
4. **Services**: Grid detailing professional offerings such as Full Stack Web Development, API Development, Performance Optimization, and Cloud Deployment.
5. **Projects**: Highlights featured work, showcasing projects like "POS System — RACPad" built with React, Node.js, PostgreSQL, and AWS.
6. **Contact** *(implied)*: To facilitate getting in touch for opportunities.

## Tech Stack

- **Framework**: [React 19](https://react.dev/) via [Vite](https://vitejs.dev/)
- **Styling**: [TailwindCSS 4](https://tailwindcss.com/) & Vanilla CSS
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Code Quality**: ESLint configuration included

## Getting Started

### Prerequisites
Make sure you have Node.js and npm installed.

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview the production build:
   ```bash
   npm run preview
   ```
